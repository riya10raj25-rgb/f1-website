# 🏎️ F1 Website

A Formula 1 fan website built with HTML, CSS, and JavaScript, served via a Node.js + Express backend.


---

## 📁 Project Structure

```
f1-website/
├── index.html       # Main webpage
├── style.css        # Styling
├── script.js        # Frontend JavaScript
├── data.json        # F1 data (drivers, teams, etc.)
├── server.js        # Express server
└── package.json     # Dependencies
```

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Hosting:** Render
